> ./FileName.py

### Flash Web Framework

#### Initiate 
**Always use with venv**
Go to the project directory and initiate venv 
> _python3 -m venv .venv_

Activate the environment (while on the directory)
> _. .venv/bin/activate_

Deactivate the environment
> _deactivate_

Install Flask on the venv
> _pip install Flask_
